# Vue Playground

Vue Playground je webová aplikácia slúžiacia na efektívne učenie sa Vue.js. Forma aplikácie spočíva v detailných popisoch a útržkoch kódu v rámci jednotlivých ukážok.

Playground si je možné pozrieť na [https://vue-playground.netlify.com/](https://vue-playground.netlify.com/)

# Prispievanie

Tento projekt je stále vo vývoji. Ak nám chcete pomôcť, prosím pozriťe si [ako začať](https://github.com/runyasak/vue-playground/blob/master/CONTRIBUTING.md). Tešíme sa na vaše efektívne pull requesty a myšlienky 😁.
